# FabsLang-Interpreter

Easy programming language you can see

| Command       | Explanation                     | 
| ------------- | ------------------------------- |
| f             | write                           |
| fa            | increase                        |
| fab           | decrease                        |
| fabs          | pointer increase                |
| fabsl         | pointer decrease                |
| fabsla        | read                            |
| fabslan       | loop start until pointer value 0|
| fabslang      | loop finish                     |
